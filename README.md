# Информационно-аналитическая система анализа влияния отзывов на репутацию

## 1. Постановка задачи

### Цель системы
Разработка прототипа информационно-аналитической системы, которая:
- Анализирует текстовые отзывы потребителей товаров и услуг
- Оценивает влияние отзывов на репутацию субъектов (компаний, брендов, организаций)
- Предоставляет аналитические инструменты для мониторинга и прогнозирования репутационных рисков

### Входные данные
- **Текст отзыва** — неструктурированный текст на русском языке
- **Числовая оценка** — рейтинг от 1 до 5 звёзд
- **Категория товара/услуги** — классификация продукта
- **Идентификатор субъекта** — уникальный ID компании/бренда

### Выходные данные
- **Класс тональности** — негативная / нейтральная / позитивная
- **Индекс репутационного влияния** — числовой показатель от -1 до 1
- **Агрегированные метрики** — по субъектам и категориям

### Практическая польза
1. **Мониторинг репутации** — отслеживание динамики отзывов в реальном времени
2. **Раннее предупреждение** — выявление негативных трендов до их эскалации
3. **Принятие решений** — обоснование маркетинговых и операционных решений
4. **Конкурентный анализ** — сравнение репутации с конкурентами

## 4. Архитектура модели

Используется **LSTM (Long Short-Term Memory)** нейросеть:
- Эмбеддинг слой для представления слов
- Двунаправленный LSTM для захвата контекста
- Полносвязные слои для классификации
- Два выхода: классификация тональности + регрессия индекса репутации

## 5. Метрики

- **Accuracy** — точность классификации тональности
- **F1-score** — сбалансированная метрика с учётом классов
- **MAE** — средняя абсолютная ошибка для индекса репутации
- **Confusion Matrix** — матрица ошибок для анализа

## 6. Ограничения прототипа

1. Синтетические данные — не отражают реальное распределение
2. Ограниченный словарь — только русский язык, базовая токенизация
3. Простая архитектура — без pre-trained языковых моделей
4. Одна модель — без ансамблирования
